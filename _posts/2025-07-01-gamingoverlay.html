Remove the 'ms gaming overlay popup' in these easy steps

Step 1: Open Registry Editor

Once you have the registry editor opened, navigate to this location inside

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\GameDVR

Look for a dword value in there called AppCaptureEnabled and set the value to 0

If this value does not exist, you need to still make sure you are within HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\GameDVR then right click and create a new dword 32 bit value and name it AppCaptureEnabled

Step 2: Reboot and it Should be Fixed
