Use this registry value to fix the missing compatibility tab.

Open up your registry editor and find this location

HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\AppCompat

If you do not have AppCompat then while having Windows selected, right click and make a new key and name it AppCompat

Next, look for a value or right click and create a new dword 32 bit value and name it DisablePropPage

A value of 1 should disable the compatibility tab while a value of 0 should show the compatibility tab again

A system restart should not be needed, you should only need to close out of your compatibility settings if you have them open, and then attempt to open them again and it should work
